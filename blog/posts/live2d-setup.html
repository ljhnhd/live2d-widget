第一次尝试 Live2D 博客装扮

2026-04-20 · 前端 Live2D

在无数次逛别人的博客、看到角落里会眨眼睛的二次元角色之后，我终于下定决心——要给自己的博客也加上一只看板娘。于是就有了今天这篇折腾记录。

什么是 Live2D Widget？

Live2D Widget 是一个开源项目，可以让你在任意网页的角落放置一个可交互的 Live2D 模型。它支持 Cubism 2 和 Cubism 5 两代 SDK，并且不依赖 jQuery 等重量级框架，非常轻量。

接入方法

最简单的方式只需一行脚本：

<script src="https://fastly.jsdelivr.net/npm/live2d-widgets@1.0.0/dist/autoload.js"></script>

把它加入页面的 <body> 末尾，刷新页面，看板娘就会出现在左下角。她还会响应你的鼠标悬浮、页面滚动等事件，说出随机台词，相当有趣。

进阶配置

项目提供了 initWidget 函数，支持多种配置项：

  • waifuPath：自定义台词 JSON 的路径
  • cdnPath：模型资源的 CDN 地址
  • modelId：默认加载的模型编号
  • tools：右侧显示的工具按钮列表
  • drag：是否允许拖拽看板娘

通过修改 waifu-tips.json，还可以让看板娘根据页面上的特定 CSS 选择器触发专属台词，非常适合博客的个性化场景。

部署心得

我最终选择直接加载 jsDelivr CDN 上的版本，零配置、零维护。如果想深度定制，可以 Fork 项目后自己修改，再通过 Cloudflare Pages 或 GitHub Pages 自行托管——这也是这个博客多页面版本正在使用的方案。

看板娘让冰冷的网页有了温度。如果你也在搭博客，不妨试试！
